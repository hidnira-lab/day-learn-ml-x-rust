# Day Learn Machine Learning with Rust ⚙️

Learning machine learning from its mathematical foundations, implemented in pure Rust. Documenting my full journey through exercises and projects.

## Goals

- Understand ML through the math first: linear algebra, calculus, and statistics, not just calling library functions.
- Implement core algorithms from scratch in Rust before reaching for crates like `ndarray`, `nalgebra`, or `linfa`.
- Learn Rust itself along the way, this is also my first time writing Rust.
- Long-term: become a contributor to the Rust ML open source ecosystem.

## Approach

Every topic follows the same loop: theory → manual calculation by hand → Rust implementation from scratch → verification against the manual result. Libraries are introduced only after the underlying mechanism is understood.

## Tech stack

- **Language:** Rust
- **Math/ML crates:** none for now, introduced gradually once the fundamentals are implemented by hand

## Status

Early stage, just getting started. This repo will grow with exercises and small projects as I progress.
